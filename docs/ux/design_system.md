# Design System (Initial Stub)

Focus: consistency, accessibility, performance.

## Principles
1. Content first.
2. Progressive enhancement.
3. Accessible defaults (WCAG AA baseline).
4. Dark-mode parity.

## Components (Planned)
- Typography scale
- Color tokens
- Buttons (primary, secondary, subtle)
- Inputs & validation
- Navigation (sidebar, breadcrumbs)
- Code blocks
